In les 4 heb ik mijn ingangschermen HTML en CSS geoptimaliseerd met behulp van ChatGPT. In les 5 heb ik dit programma ontwikkeld. In les 6 heb ik verder gewerkt aan Netlify, Visual Code en Github. Ik heb mijn treinschermen omgezet in een scherm voor de telefoon en heb ik deze projectdocumentatie verder aangepast.
